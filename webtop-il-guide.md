## 💻 **דרך 2: התקנה ידנית (מתאים למתקדמים)**

### **מה אתה צריך:**
- ✅ גישה למחשב עם Python 3
- ✅ יכולת להריץ פקודות

### **מה לעשות:**

**שלב 1:** שכפל את המערכת:
```bash
git clone https://github.com/shaike1/webtop-skill.git
cd webtop-skill
```

**שלב 2:** הרץ את המתקין:
```bash
./install.sh
```

**שלב 3:** ערוך את הקובץ `.env`:
```bash
nano .env
```

**שלב 4:** הכנס את הפרטים:
```bash
# שם משתמש וסיסמה
WEBTOP_USERNAME=השם_משתמש_שלך
WEBTOP_PASSWORD=הסיסמה_שלך
WEBTOP_DATA_KEY=ה_DATA_KEY_שמצאת

# הילדים שלך (שם,כיתה)
WEBTOP_CHILDREN=יובל,2-5,שירה,4-3

# WhatsApp (אופציונלי)
WHATSAPP_TARGET=972503219663
```

**שלב 5:** התחל:
```bash
python3 homework_monitor.py
```

## 💡 **טיפים חשובים**

### ✅ **מה עובד מצוין:**
- שיעורי בית מגיעים כל בוקר
- תזכורות למבחנים
- עדכונים ל-Google Calendar
- הודעות WhatsApp יפות

### ⚠️ **מה לשים לב:**
- ה-DATA KEY הוא הכי חשוב - בלי זה לא עובד
- צריך להתחבר ל-WEBTOP לפני שמעתיקים
- ה-Data Key משתנה לפעמים - אם לא עובד, מעתיקים שוב

### 🆘 **יש בעיה?**

**❌ "לא מצליח להתחבר"**
→ תבדוק את שם המשתמש והסיסמה
→ תוודא שה-DATA KEY נכון
→ תנסה להתחבר דרך האתר

**❌ "אין שיעורי בית"**
→ WEBTOP לא תמיד מעדכן
→ תנסה מאוחר יותר
→ תבדוק אם יש שיעורים באתר

**❌ "WhatsApp לא עובד"**
→ תבדוק את הגדרות ה-WhatsApp
→ תוודא שהקבוצה קיימת
→ תנסה לשלוח הודעה ידנית
